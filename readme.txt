== Description ==

Email Alert subscribe form. You can manage through the admin pages of your blog. Simple put the shortcode '[sh_email_alert_form]' into your page or post.

== Installation ==

Installing the SH Slideshow plugin is quick and simple:

1. Begin by downloading the plugin file (ZIP)
1. Extract the files to your local machine
1. Upload the whole sh-email-alert folder (including the folder itself) to your plugins directory - typically http://www.yourdomain.com/wp-content/plugins/
1. Activate the plugin using your Wordpress admin pages
1. Add the form code to your theme

== Frequently Asked Questions ==
